There are 5 csv files of arbitrary data to input as examples.
stars.csv - contains many stars for file input of many objects (can work for any branch 1 class)
star_data.csv - contains experimental (vector) star data for file input (can work for any planet as well)
galaxies.csv - contains many galaxies for file input of many objects (can work for any branch 1 class)
galaxy_data.csv - contains experimental (vector) galaxy data for file input 
cepheids.csv - contains many cepheids for file input of many objects (can work for all branch 2 classes)
